===========
Tile client
===========

The tile-client extension contains the framework for handling different tile servers,
and implements the logic for retrieving tiles from Bing and OpenStreetMap.


.. toctree::
   :maxdepth: 1
   
   tileclient


**Maven**::
   
    <dependency>
      <groupId>org.geotools</groupId>
      <artifactId>gt-tile-client</artifactId>
      <version>${geotools.version}</version>
    </dependency>
