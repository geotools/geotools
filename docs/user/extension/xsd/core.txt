XSD-Core
--------

These extensions are used by the gt-xml module to handle basic xml schema types
and xlink.

References:

* :doc:`gt-xml </library/xml/index>`
* http://www.w3.org/TR/xmlschema-1/
* http://www.w3.org/TR/xlink/
