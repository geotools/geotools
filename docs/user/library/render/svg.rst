SVG Plugin
----------

The **gt-svg** plugin uses the Batik library to support the use of SVG graphics.

**Maven**::
   
    <dependency>
      <groupId>org.geotools</groupId>
      <artifactId>gt-svg</artifactId>
      <version>${geotools.version}</version>
    </dependency>
