Allows the EPSG database to be used with the HSQL Java database as an Authority for GeoTools CRS generation. At the difference of epsg-access and epsg-postgresql, the epsg-hsql plugin contains a copy of the EPSG database. This plugin can work "out of the box" without any specifial action on the user side.
